Timestamp Microservice

Welcome to Luke's Timestamp microservice!

In the URL, add /api/:date in order to get back the date in utc and unix format

To get the current date, simply add /api to the url

Example Usage:

  • /api/2015-12-25
  • /api/1451001600000

Example Output:

{"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}
